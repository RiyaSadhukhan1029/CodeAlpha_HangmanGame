# Hangman Game 🎮

This is a simple Hangman Game built using Python.

## Features:
- Random word selection
- 6 attempts limit
- User input-based guessing

## Concepts Used:
- random module
- loops (while)
- if-else conditions
- lists and strings

## How to Run:
1. Run the file using Python
2. Guess letters to find the word
